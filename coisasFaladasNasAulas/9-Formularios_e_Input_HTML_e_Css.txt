Os elementos <fieldset> e <legend>

O elemento <fieldset> é uma maneira conveniente de criar grupos de widgets que compartilham o mesmo propósito. Também é utilizado para agrupar os campos de um formulario. Um elemento <fieldset> pode ser rotulado com um elemento <legend>. O elemento <legend> descreve formalmente a finalidade do elemento <fieldset> ao qual está incluído.

 É estritamente proíbido aninhar um formulário dentro de outro formulário. Isto pode fazer com que os formulários se comportem de maneira imprevisível baseada no navegador que está sendo utilizado
